# توثيق تحليل الذكاء الاصطناعي للأنشطة

يحتوي هذا الملف على تفاصيل البيانات التي يتم إرسالها لنموذج الذكاء الاصطناعي (Gemini) لتوليد الملخص الذكي لكل نشاط.

## البيانات المستخدمة في التحليل

يتم جمع البيانات التالية من قاعدة البيانات لكل نشاط:

- **اسم النشاط (`name`)**: لتحديد موضوع التقرير.
- **وصف النشاط (`description`)**: لشرح تفاصيل المهام المنفذة.
- **التكلفة الإجمالية (`cost`)**: لتقييم الجدوى الاقتصادية.
- **إحصائيات الوصول**:
    - عدد المجموعات المستفيدة (`beneficiaries_count`).
    - عدد الطرود أو الوحدات الموزعة (`parcels_count`).
- **مؤشرات الجودة**:
    - متوسط التقييم الرقمي للنشاط (`average_rating`).
    - عينة من تعليقات المشاركين (`feedbacks`): يتم إرسال آخر 5 تعليقات مسجلة.

## صياغة الطلب (Prompt Logic)

يتم إرسال هذه البيانات مع تعليمات برمجية تطلب من الذكاء الاصطناعي:

1. كتابة **ملخص تنفيذي احترافي** باللغة العربية.
2. التركيز على **الأرقام والإنجازات** المحققة.
3. اعتماد أسلوب **رسمي موجه للإدارة**.
4. عدم ذكر أسماء أشخاص والتركيز فقط على النتائج العامة.
